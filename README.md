#  Bibliothèque en Ligne
Site web complet de gestion de bibliothèque en ligne, développé de façon entièrement autonome. 
Le projet couvre l'intégralité du cycle de développement : conception de l'architecture, 
développement front-end (HTML, CSS, JavaScript), mise en place d'une API REST avec Node.js 
et Express.js, gestion des données via une base de données relationnelle SQLite, 
authentification sécurisée des utilisateurs (hashage bcrypt, tokens JWT), 
et mise en production du site.

##  Fonctionnalités

- Inscription et connexion sécurisées (bcryptjs + JWT)
- Catalogue de livres avec recherche et filtre par genre
- Emprunt et retour de livres en ligne
- Page profil avec historique des emprunts
- Base de données SQL (SQLite)
